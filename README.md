# GeoAcademyTracker

A comprehensive tool for tracking and managing geographical academy data.

## Description

GeoAcademyTracker is designed to help users search, manage, and analyze geographical academy information efficiently. The tool provides an intuitive interface for accessing academy-related data with robust search capabilities.

## Features

- **Search Functionality**: Type (2) to initiate search operations
- **File Management**: Easy navigation through project files
- **Preview Capabilities**: Quick preview of file contents and metadata
- **Commit Tracking**: Monitor project changes and version history
