# GeoAcademyTracker

## Overview
GeoAcademyTracker is a dedicated web platform designed to streamline customer onboarding and training management for the Geography team. This tool integrates with HubSpot to automatically track customer engagement and prompt trainers to provide Geo Academy access.

## HubSpot Integration & Workflow Automation

### Customer Tracking
- **Automatic Visitor Monitoring**: HubSpot tracks all website visitors, capturing geographic data, page engagement, and interaction patterns
- **Contact Profile Creation**: Each visitor is automatically created as a HubSpot contact with their activity history
- **Engagement Scoring**: System assigns engagement scores based on time spent, pages visited, and repeat sessions

### Trainer Prompt System
- **Automated Access Provisioning**: When customers meet engagement thresholds, the system automatically notifies trainers via HubSpot tasks and email alerts
- **Trigger-Based Workflows**: 
  - High engagement (5+ minutes on site) → Auto-prompt for Geo Academy access
  - Multiple return visits → Priority follow-up required
  - Specific page interactions → Targeted training recommendations
- **One-Click Access Provision**: Trainers can directly provision Geo Academy access from HubSpot contact records

## Key Sections & Functionality

### 1. Customer Dashboard
- Real-time view of active visitors and their geographic locations
- Engagement metrics and interaction history
- Priority customers flagged for immediate trainer attention

### 2. Training Progress Tracking
- Monitor customer progression through Geo Academy modules
- Completion rates and knowledge assessment results
- Automated reminders for stalled learners

### 3. Automated Alert System
- Instant notifications when customers are ready for academy access
- Escalation prompts for unaddressed high-engagement leads
- Performance analytics on trainer response times

### 4. Geographic Analytics
- Regional engagement heat maps
- Location-based training effectiveness
- Territory management insights

## How Account Managers Use This Tool

### Daily Workflow
1. **Monitor Active Engagement**: Review dashboard for high-engagement customers
2. **Respond to Automated Prompts**: Act on HubSpot tasks for academy access provisioning
3. **Track Training Progress**: Follow customer progression through assigned modules
4. **Generate Reports**: Export engagement and completion metrics for performance review
